# 烛龙 ZhuLong · 安装与首次运行

## 1. 系统要求

- Windows 10/11 64 位
- MetaTrader 5（已登录券商账户或模拟盘）
- **Python 3.10+**（安装时勾选 *Add Python to PATH*）

---

## 2. 同事电脑怎么部署（推荐）

### 方式 A：安装包（推荐，MT5 自动部署）

1. 从发布目录拿到 **`output\ZhuLong_Setup_v3.1.0.exe`**
2. **先安装并登录 MT5**（若尚未安装）
3. 双击 Setup，按向导完成（默认 `C:\Program Files\ZhuLong\`）
4. 安装程序会自动：
   - 安装 WinUI 3 运行时、VC++ 运行库
   - 运行 `scripts\deploy-mt5-indicator.ps1`，把 **DLL + 指标** 复制到本机所有 MT5 目录
5. 安装完成后执行 **§3 Python 依赖** 与 **§5 首次运行**

> 若安装时 MT5 尚未安装，安装不会失败；MT5 装好后再从开始菜单点 **「部署 MT5 指标」** 即可。

### 方式 B：拷贝文件夹（无安装包时）

将整个 **`publish\win-x64`** 目录复制到同事电脑，例如 `C:\Program Files\ZhuLong\`，然后：

```powershell
cd "C:\Program Files\ZhuLong"
.\install_python_deps.ps1
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\deploy-mt5-indicator.ps1 -InstallDir . -StopMt5First
```

## 3. 需要复制哪些文件

### 给同事：只发安装包

| 文件 | 说明 |
|------|------|
| `ZhuLong_Setup_v3.1.0.exe` | 唯一必需，内含程序、模型、Python 脚本、MT5 桥接文件 |

### 手动拷贝 `publish\win-x64` 时：整个目录

关键子目录：

| 路径（安装目录下） | 用途 |
|-------------------|------|
| `ZhuLong.exe` + 所有 `.dll` | 主程序 |
| `mql5\Libraries\ZhuLongMt5Pipe.dll` | **MT5 管道 DLL**（必须部署到 MT5） |
| `mql5\ZhuLongIndicator.mq5` | 指标源码（当前 **v1.16**） |
| `mql5\ZhuLongIndicator.ex5` | 预编译指标（安装包构建时生成） |
| `indicators\` | 同上 mq5/ex5 的副本 |
| `models\` | 推理模型 |
| `zhulong\` | Python 推理引擎 |
| `scripts\deploy-mt5-indicator.ps1` | MT5 一键部署脚本 |
| `install_python_deps.ps1` | Python 依赖安装 |

**不要只复制 mq5** — 没有 `ZhuLongMt5Pipe.dll` 会出现 `cannot load [126]`，管道无法连接。

## 4. MT5 连接失败 / 指标与 DLL

### 最新指标位置（源码）

- 开发仓库：`mql5\ZhuLongIndicator.mq5`（**v1.16**）
- 安装目录：`C:\Program Files\ZhuLong\mql5\ZhuLongIndicator.mq5`
- 安装目录副本：`C:\Program Files\ZhuLong\indicators\ZhuLongIndicator.mq5`

部署成功后，MT5 侧路径为：

```
<MT5数据目录>\MQL5\Libraries\ZhuLongMt5Pipe.dll
<MT5数据目录>\MQL5\Indicators\ZhuLongIndicator.ex5
<MT5数据目录>\MQL5\Indicators\ZhuLongIndicator.mq5
```

`<MT5数据目录>` 可能是：

- `C:\Program Files\WCG Group MT5 Terminal\`（便携版）
- `%APPDATA%\MetaQuotes\Terminal\{32位十六进制ID}\`

### DLL 如何「导入」

MT5 **不会**在界面里手动选 DLL。流程是：

1. 把 `ZhuLongMt5Pipe.dll` 放到 `<MT5>\MQL5\Libraries\`
2. MT5 → **工具 → 选项 → EA交易** → 勾选 **「允许 DLL 导入」**
3. **重启 MT5**
4. 在 XAUUSD **M1** 图表挂载 `ZhuLongIndicator`（导航器 → 指标 → 自定义）
5. 查看「专家」日志：不应再出现 `cannot load 'ZhuLongMt5Pipe.dll' [126]`

### 常见「无法连接」日志

| MT5 专家日志 | 原因 | 处理 |
|-------------|------|------|
| `cannot load 'ZhuLongMt5Pipe.dll' [126]` | Libraries 下无 DLL | 运行「部署 MT5 指标」或 deploy 脚本 |
| `unresolved import function call` | 同上，DLL 未加载 | 同上 + 重启 MT5 |
| `无法连接 ZhuLong_Data 管道` | 烛龙未运行或未点连接 | 启动 ZhuLong.exe → **连接 MT5** |
| 烛龙显示「绘图管道未连接」 | 图表未挂指标或 MT5 未重启 | 重新加载 ZhuLongIndicator |

### 手动重部署

```powershell
cd "C:\Program Files\ZhuLong"
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\deploy-mt5-indicator.ps1 -InstallDir . -StopMt5First
```

日志：`%LOCALAPPDATA%\ZhuLong\mt5_deploy.log`

## 6. 首次运行

1. 启动 **烛龙 ZhuLong**
2. 点击 **连接 MT5**
3. 点击 **开始运行**
4. 确认日志：**数据管道已连接**、**绘图管道已连接**、有 M1 数据

验收测试信号（可选）：

```powershell
# 写入 %LOCALAPPDATA%\ZhuLong\inject_draw.json
{"action":"draw_signal","signal_id":"TEST","symbol":"XAUUSD","direction":"buy","entry":4200,"sl":4190,"tp":4220,"confidence":"0.90","expiry_minutes":240}
```

## 7. 持仓管理说明

烛龙 **直接托管信号**（虚拟持仓），**不需要**在 MT5 下单或匹配 Comment。  
止损/平仓后图表信号会自动清除（v1.16+）。

## 10. 故障排查

| 现象 | 处理 |
|------|------|
| Python 初始化失败 | 确认 `py -3` 可用，重跑 `install_python_deps.ps1` |
| 管道无 M1 | 部署 DLL + 挂指标 + 允许 DLL 导入 + 重启 MT5 |
| 止损后图表不清除 | 确认指标 v1.16+，烛龙与 MT5 均已重启 |
| 与旧版冲突 | 运行 `mql5/Scripts/ZhuLong_ClearLegacyDash.mq5` |
