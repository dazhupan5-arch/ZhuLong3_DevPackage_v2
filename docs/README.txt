烛龙 ZhuLong 量化交易系统 - 快速说明
=====================================

1. 安装 ZhuLong_Setup.exe
2. 复制 indicators\ZhuLongIndicator.mq5 到 MT5 的 MQL5\Indicators\ 并编译
3. XAUUSD、USOIL 的 M1 图表各挂载指标
4. 启动 ZhuLong.exe → 连接 MT5 → 开始运行
5. 图表可见 ATR 通道 + EMA 线；信号出现后手动下单，Comment 粘贴 ZhuLong_<signal_id>

详细说明见同目录 UserGuide.md 或联系开发者 Stephen.Pan
